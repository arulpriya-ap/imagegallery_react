# 🦁 WildFrame

WildFrame is a responsive React-based animal gallery that showcases a collection of wild animals with beautiful images, names, and short descriptions. The project is designed to provide an engaging browsing experience while demonstrating React component architecture, props, and responsive layouts.

---

## 📌 Features

- 🐾 Display a collection of wild animals.
- 🖼️ Responsive image gallery.
- 📱 Mobile, tablet, and desktop friendly.
- ✨ Hover animation on animal cards.
- 📌 Sticky header while scrolling.
- ⚛️ Built using reusable React components.
- 🎨 Clean and modern UI design using CSS Grid and Flexbox.
